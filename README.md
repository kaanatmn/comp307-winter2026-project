# COMP 307 Winter 2026 Project: McGill Booking App

## Tech Stack
* Frontend: React + Vite
* Backend: Java + Spring Boot
* Database: PostgreSQL (via Docker)
* Auth: JWT + Spring Security

## Quick Start
1. Run \docker-compose up -d\ from the root to start the database.
2. CD into \backend\ and run your Spring Boot app.
3. CD into \frontend\, run \
pm install\, then \
pm run dev\.
